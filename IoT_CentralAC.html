IoT Central AC Control (Pottery Storage)

An IoT system to monitor and regulate temperature & humidity in pottery storage using cloud telemetry and edge microcontrollers.

Telemetry, rules engine, and device management for safe storage.

Summary

Designed an IoT system that uses edge sensors and a cloud dashboard to regulate conditions in pottery storage — prevents cracking through proactive humidity and temperature control.

Skills

  • Raspberry Pi / ESP32
  • MQTT / IoT Central
  • Cloud dashboards
  • Embedded sensing & control

System Overview

Edge devices read DHT/DS18B20 sensors and publish via MQTT to a cloud broker, where rules maintain safe setpoints and notify users on deviation.
